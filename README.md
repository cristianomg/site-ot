# 💀 Dungeon Souls - Ultra Dark OT Server 💀

Um website com tema ultra-sombrio das masmorras sombrias, com efeitos de tempestade e atmosfera das trevas, para servidores Open Tibia.

## 🏰 Características do Layout

### 🎨 Visual Ultra-Dark das Masmorras
- **Tema de cores**: Laranja-vermelho das forjas sombrias (`#ff6b47`, `#8b0000`) contrastando com fundos preto puro (`#000000`, `#0a0a0a`)
- **Fontes**: Cinzel e Cinzel Decorative (Google Fonts) para aparência sinistra das masmorras
- **Background**: Gradiente ultra-escuro com partículas sombrias (💀, 🔥, ⚔️) e tempestades
- **Elementos decorativos**: Ícones do universo das masmorras (💀, 🔥, ⚔️, ⚡, 🌑) com brilho vermelho-laranja
- **Atmosfera**: Tempestades das masmorras com as almas perdidas vigiando

### ✨ Efeitos Interativos

#### 🌟 Partículas Sombrias das Masmorras
- Partículas sombrias (💀, 🔥, ⚔️, ⚡, 🌑) representando o poder das almas perdidas
- Efeito de neve ocasional (❄️) reduzido para dar espaço às tempestades
- Animações sinistras mais espaçadas para criar atmosfera de vigilância

#### 🌧️ Sistema de Chuva Medieval Épica
- **Tempestades realísticas**: Gotas de chuva que caem verticalmente pela tela
- **Múltiplas ondas**: Cada tempestade possui 4 ondas consecutivas de chuva
- **Timing dinâmico**: Primeira chuva após 20 segundos, depois a cada minuto
- **40% de chance**: Sistema probabilístico para tornar as tempestades surpresa
- **30 gotas por onda**: Chuva intensa com gradiente azul translúcido
- **Animação fluida**: 1.5 segundos de duração com rotação natural

#### ⚔️ Efeitos Sombrios nos Botões
- Efeitos dos "Olhos das Trevas" (💀) nos botões primários
- Efeitos visuais de fogo (🔥) quando clicados
- Brilho vermelho-laranja dinâmico simulando o poder das trevas

#### 🎭 Sistema de Toasts Medievais Sombrios
- Frases motivacionais medievais que aparecem a cada 45 segundos
- "Oráculo Medieval" com fundo ultra-escuro e bordas douradas
- Animações suaves de entrada e saída com blur effect intensificado

#### 📊 Contador Dinâmico
- Número de jogadores online se atualiza automaticamente
- Variação realista para simular atividade do servidor

### 🗡️ Easter Eggs

#### 🎮 Konami Code Medieval
- Digite: ↑↑↓↓←→←→BA
- Ativa "Modo Épico" com:
  - Explosão de 50 espadas douradas (⚔️)
  - Filtro visual especial por 10 segundos
  - Mensagem de "bonus" de XP

- **Mobile-first**: Layout adaptável para todas as telas

### 🔧 Tecnologias Utilizadas

- **ASP.NET Core 8.0**: Framework web principal
- **Bootstrap 5**: Sistema de grid e componentes base
- **Google Fonts**: Cinzel e Cinzel Decorative
- **CSS3**: Animações e efeitos avançados
- **JavaScript ES6+**: Interatividade e efeitos dinâmicos

### 🚀 Como Executar

```bash
# Navegar para o projeto web
cd src/OtServer.Web

# Executar o projeto
dotnet run
```

O site ficará disponível em: `https://localhost:5001` ou `http://localhost:5000`

### 🎯 Inspiração

Design inspirado em temas sombrios transformado em tema ultra-dark baseado no universo das masmorras perdidas, com foco em:
- **Atmosfera das Masmorras**: Reproduz a sensação sinistra das profundezas sombrias
- **Almas Perdidas**: Elementos visuais que remetem às almas que vagam pelas trevas
- **Poder das Trevas**: Interface que transmite corrupção e poder sombrio
- **Experiência das Trevas**: Imersão completa no universo sombrio das masmorras
- **Sensação de estar nas Profundezas** observando as masmorras durante uma tempestade
- **Cores de fogo e lava**: Paleta inspirada nas forjas das trevas e chamas das almas
